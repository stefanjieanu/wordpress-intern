== Description ==

####[Slideshow for Wordpress](http://2joomla.net/wordpress-plugins/2j-slideshow) - it's fully responsive slideshow with swipe and powerful mobile devices support. Slideshow have classic design with clean and stylish interface elements. Slideshow is highly customizable and support fullscreen mode. Slide of the slideshow could contain not only images, but also videos and HTML based description of the images.
You can insert this slideshow to any Wordpress content article. You can insert as much slideshows to the content article as you wish. Every slideshow can be configured with individual parameters. You can use plugin tags wizard for creation plugin tags.
All front end interface elements are highly customizable. For installation and configuration you don't need any special skills everything build in Wordpress native style. For upload of the image you can use build in images manager with advanced functionality for upload, sorting and customizing of the slideshow images.

== Installation ==

Just install it using regular Wordpress plugins installing mechanism. You can download of the 2J Slideshow installer from directory and install it manually in wordpress plugin manager in admin section of your blog. Another way install it in plugins manager without download of the archive to your computer, just click install button in plugins manager and activate plugin after that. If you have some questions related with our plugin feel free to contact our support team we happy to help you!

== Frequently Asked Questions ==

= Can I create slideshows with different settings on the same page? =

Yes, it's possible to define different settings for every slideshow. You can customize style and inserface elements of every slideshow different way and insert them into the same page or post. As result you'll have multiply slideshow on page with different settings.

= How to upload images into slideshow? =

For uploading images into slideshow implemented advanced slides manager. Where you can upload images one by one or in batch mode.

= Is it possible to publish video play button on thumbnails of the slideshow? =

No, it's not possible to publish video play button on slideshow thumbnails.

= How to insert slideshow into page? =

You can insert slideshow into page with shortcode, which you can generate when slideshow created in slideshow manager.

= Is it possible to insert multiply slideshows on page ? =

Yes, you can insert as much slideshows on page as you need. Every slideshow could have different settings.

= How to install Slideshow plugin? =

Just install it using regular Wordpress plugins installing mechanism.  If you have some questions related with our plugin feel free to contact our support team we happy to help you!

= Which navigation elements implemented in slideshow? =

In our slideshow you can use navigation arrows, thumbnails, dots.

= Is it possible insert videos to the images of the slideshow? =

Yes, it's possible to insert video links in every image. You can define link to Youtube or Vimeo videos.

= Is it possible to enable swipe images in slideshow? =

Yes, functionality for support swipe and touch screen devices in slideshow is optional. So you can turn on/off this functionality in slideshow backend.

= How to edit uploaded images in slideshow? =

In slides manager you can find additional options which make you able to rotate, cut or align images uploaded to the slideshow.

= How to insert slideshow into post? =

You can insert slideshow into post with shortcode, which you can generate when slideshow created in slideshow manager.

= It there any limits for slideshow images size? =

No, we don't have any limits for images size. Size of the images could be limited by server settings.

= Is it possible to switch between thumbnails slideshow navigation elements and dots slideshow menu ? =

yes, you can switch between two slideshow slides navigation modes: thumbnails or dots menu. Also you can disable slideshow navigation menu at all.

= Is it possible to insert images slideshow into page? =

yes, it's possible insert sllideshow into page.

= Is it possible to publish video play button on images of the slideshow? =

Yes, it's not possible to publish video play button on slideshow images.

= Is it possible to upload multiply slideshow images at the same time ? =

Yes, our slides manager in slideshow support batch upload functionality. You can select multiply images and upload them to the slideshow by one session.

= Is it possible to sort slideshow images after upload ? =

Yes, inside slides manager you can sort ordering of your uploaded images in slideshow.

= Where to get slideshow shortcode? =

You can find shortcode inside slideshow settings section. When you open slideshow for edit you'll see big button with Get shortcode label. When you click on this label it's show you popup with shortcode for current slideshow.

= Is it possible to insert images slideshow into post? =

yes, it's possible insert sllideshow into post. You can use slideshow shortcode which could be copied form the settings section or from the slideshow listing. When you open post or page for edit you can insert there shortcode, save and check results on the front end.
